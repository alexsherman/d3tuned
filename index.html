to create a string, simply click and drag

the longer the string, the higher the pitch

if you'd like to be more precise, look at the bottom of sidebar to see the length of the string you're creating

for more information about pitch, intervals, and ratios, click here


to change the volume, release time, wave, or color, use the buttons to the side


to enter delete mode, click backspace

while in delete mode, plucking a string will delete it instead of creating a sound

to exit delete mode, click backspace again


to locally save your work, click save

a small json file should automatically download


to load something you've save, click load and find the json file
